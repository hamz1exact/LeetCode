# 2651. Calculate Delayed Arrival Time

Difficulty: Easy
Priority: High
Topic: Math
Time Complexity: O(n)
Space Complexity: O(1)

$$
Solution
$$

```python
class Solution:
    def findDelayedArrivalTime(self, arrivalTime: int, delayedTime: int) -> int:
        return (arrivalTime+delayedTime)%24
```
